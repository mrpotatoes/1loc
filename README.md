# Favorite single line of code

What's your favorite JavaScript single LOC (line of code)? Let me know!

## Running it on local

```shell
git clone git@github.com:mrpotatoes/1loc.git
cd 1loc
npm install
npm run start
```

Open http://localhost:8081 in your browser to see it in action.
